What is difference between event.stopPropagation and event.preventDefault

stopPropagation() method prevents further propagation of an event during event flow. It means that this method prevent capturing or bubbling. Execute only first event in stack of events.

preventDefault() method use for prevent default event for target element such as visit link on click on a element. The default action will not occur.
